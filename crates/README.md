# Rust Workspace Crates

Reusable Rust libraries live here.

- `sentinel-common/`: shared types, errors, traits, and utilities
- `sentinel-config/`: config loading, validation, and overrides
- `sentinel-telemetry/`: packet, host, and service telemetry ingestion
- `sentinel-flow/`: flow tracking and protocol session state
- `sentinel-detection/`: signature, heuristic, and anomaly execution
- `sentinel-native-bridge/`: Rust-side contracts for C, C++, and ASM layers
- `sentinel-policy/`: confidence scoring, risk weighting, and action policy
- `sentinel-response/`: safe containment and rollback actions
- `sentinel-forensics/`: timelines, evidence packaging, and audit records
- `sentinel-storage/`: durable storage interfaces and query paths
- `sentinel-bio-response/`: host health and resource-awareness logic
- `sentinel-scenario/`: scenario definitions and replay contracts
