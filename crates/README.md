# Rust Workspace Crates

Reusable Rust libraries live here.

- `crystal-common/`: shared types, errors, traits, and utilities
- `crystal-config/`: config loading, validation, and overrides
- `crystal-telemetry/`: packet, host, and service telemetry ingestion
- `crystal-flow/`: flow tracking and protocol session state
- `crystal-detection/`: signature, heuristic, and anomaly execution
- `crystal-policy/`: confidence scoring, risk weighting, and action policy
- `crystal-response/`: safe containment and rollback actions
- `crystal-forensics/`: timelines, evidence packaging, and audit records
- `crystal-storage/`: durable storage interfaces and query paths
- `crystal-bio-response/`: host health and resource-awareness logic
- `crystal-scenario/`: scenario definitions and replay contracts
