# Test Suites

Validation is split by outcome type so regressions are easier to isolate.

- `detection/`: signal and rule correctness
- `integration/`: end-to-end system behavior
- `performance/`: latency and throughput checks
- `regression/`: fixed bug coverage
- `replay/`: PCAP and scenario replay validation
- `safety/`: rollback, bounded-response, and false-positive checks
